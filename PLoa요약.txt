Lost Ark AI 기반 데이터 분석 플랫폼(PLoa)

시세 확인, 스펙 판단, 수집 진행 등을
분석하여 유저의 시간 절약, 재화 효율성을
향상시키는 유틸리티 사이트

게임 내 데이터를 통합 분석하고,
직관적인 UI로 정보를 제공

사용자: 로스트아크 유저
- 게임 내 실시간 시세, 캐릭터 스펙, 컨텐츠 효율 등 다양한 데이터를 빠르게 확인하고 싶은 니즈

핵심 요구사항: 시세 크롤링, 빠른 화면 구성,
사용자 커스텀 화면 등을 직관적인 UI로 구현

프론트엔드: React 기반 SPA (Tailwind CSS + Shadcn/UI)
백엔드: Spring Boot 기반 API 프록시 및 데이터 가공 · 분석 서버

구조: 프론트 → Spring Boot → 데이터 수집 → 분석 및 시각화 → 클라이언트 렌더링

기능별 탭 구조 UI: 프로필, 장비, 각인, 카드, 수집, 시세 등 카드형 정보 표시


인게임 관련 정보 조회는 API 키를 등록하여 진행
(현재 개인용으로 사용량 제한, 추후 사이트 서비스 시 게임사에 제한 확장 요청해야함)

캐릭터 정보 조회 : 프로필/장비/각인/스킬/보석 등 카드형 UI 출력
수집 컨텐츠 추적 : 수집 컨텐츠 진행도 시각화 및 지도 기반 지역별 분석
클래스 빌드 통계 : 직업별 스킬, 각인, 장비 조합 분석 → 인기 순위 및 비율 통계 시각화
골드 수익 계산기 : 주간 컨텐츠 기준 수익 예상치 산출
마리샵 분석 : 실시간 상품 가치 평가 → 골드 대비 효율 높은 상품 추천
스펙업 추천 : 캐릭터 데이터 기반으로 강화, 보석, 장비 ROI 분석 및 우선순위 제시
골드 시세 분석 : 디스코드 거래소 채팅 기반 실시간 시세 수집 및 그래프 시각화
시각화 대시보드 : 시세, 수익, 추천 등 모든 데이터를 통합해 대시보드 형태로 출력(사용자 커스텀)
알림/구독 시스템 : 조건부 실시간 알림 + 디스코드 웹훅 + 소리 효과 포함
관리자 페이지 : 데이터 모니터링, 알림 설정, 컨텐츠 등록 및 유저 분석 기능 포함
다크 모드 : 기본/다크 모드 전환
공지사항 : 인게임 공지사항(API로 받아옴)과 사이트 공지사항 안내

1차 목표는 캐릭터 정보 검색 + 계정 수집 컨텐츠 요약 및 시각화 구현된 MVP(Minimum Viable Product) 완성
이후 재화 시세 시각화, 빌드 통계, 골드 계산기 등 고도화 기능을 점진적으로 통합할 예정
최종적으로는 사용자가 전략적으로 스펙을 강화하고 골드를 효율적으로 사용할 수 있는
'게임 파트너 플랫폼'으로 확장
향후 Discord 챗봇 통합, 모바일 대응, 빌드 공유 기능 등 기능 확장도 고려
